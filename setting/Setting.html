🔧 วิธีรัน Backend
ต้องรัน scraper.py ก่อนเพื่อให้เว็บดึงข้อมูลได้จริง

1. ติดตั้ง dependencies:
pip install flask flask-cors requests beautifulsoup4 pandas openpyxl

2. รัน:
python scraper.py

3. เปิดเว็บ index.html แล้วกด Scrape Now

ถ้าไม่รัน backend เว็บจะแสดงข้อมูลตัวอย่าง (Demo mode) แทน
💾 จัดการข้อมูล

ข้อมูลถูกเก็บใน localStorage ของ Browser และไฟล์ db.json
